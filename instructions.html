Instructions:

Note: This is designed to be more of an electronic whiteboard than a faults database. It should be quick and easy to use as sitesDB entry is still required.

To add a fault the site must already exist. Click on sites > Add site to add a site

To edit a fault click on the fault number in the fault# column on the home page

To remove a fault edit the fault and set it to cleared

Warning! If you edit a site number or name it will cascade through the database.
-All faults with this name or number will be updated.

Sites can be edited via the sites drop-down

If you get the error page it may be that the site number or name already exists

Please let me know if you get an error! Maybe add it to faultsDB!
